# my_prompt

A bash prompt with the username, the current working directory and the git branch as well as the statuses.
These are the symbols for the git statuses:

synced = ""
dirty & synced = *
unpushed = △
dirty & unpushed = ▲
unpulled = ▽
dirty & unpulled = ▼
unpushed & unpulled = ⬡
dirty, unpushed & unpulled= ⬢
